## 🚀 About Me

I am a **Full Stack Developer** and **QA Specialist** based in Jakarta, Indonesia. I specialize in developing high-performance web applications and ensuring their production readiness through thorough manual testing and QA methodologies. I love translating complex business requirements into scalable code and polished user interfaces.

- 🔭 **Currently Working on:** Enhancing full-stack system architecture and deep-diving into automated testing.
- 💼 **Experience with:** Enterprise applications at **Telkomsel**, **PT Astra Daihatsu Motor**, and **PT Milenia Mega Mandiri**.

- 🤝 **Collaboration:** Open to collaborate on open-source projects or remote full-stack roles.
